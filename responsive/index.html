Thiết kế Web Phản Hồi (Responsive Web Design)

Thiết kế web phản hồi là kỹ thuật phát triển trang web sao cho giao diện và bố cục của trang tự động điều chỉnh dựa trên kích thước màn hình của thiết bị người dùng, từ máy tính để bàn đến điện thoại di động.

Media Queries

Media queries là một phần quan trọng của thiết kế web phản hồi. Nó cho phép trang web thay đổi phong cách CSS dựa trên các yếu tố như chiều rộng, chiều cao của màn hình, và các thuộc tính khác của thiết bị. Ví dụ, bạn có thể thay đổi bố cục từ một dạng hiển thị trên máy tính sang một dạng hiển thị dễ đọc hơn trên thiết bị di động.

Dưới đây là một ví dụ về cách sử dụng media queries trong CSS:

                
                    /* Máy tính để bàn */
                    @media (min-width: 1024px) {
                        .container {
                            width: 80%;
                        }
                    }

                    /* Máy tính bảng */
                    @media (max-width: 768px) {
                        .container {
                            width: 100%;
                        }
                    }

                    /* Điện thoại di động */
                    @media (max-width: 480px) {
                        .container {
                            width: 100%;
                            padding: 10px;
                        }
                    }

Tại sao Media Queries quan trọng?

Media queries giúp trang web của bạn dễ dàng truy cập trên mọi thiết bị, giúp cải thiện trải nghiệm người dùng và tối ưu hóa hiệu suất.
